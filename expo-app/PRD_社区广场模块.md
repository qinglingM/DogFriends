# 社区广场模块 PRD（含 UI/UX 定义）

## 1. 模块概述

* **模块名称**：社区广场
* **模块目的**：为狗主人提供分享遛狗日常、求助经验、狗狗配对、好物推荐等社区交流空间
* **支持格式**：文字、图片（3:4）、视频
* **用户互动行为**：点赞、评论、收藏

---

## 2. 功能清单

| 功能 | 描述 | 用户动作 | UI/UX 说明 |
| --- | --- | --- | --- |
| 帖子发布 | 用户可以发帖分享经验或求助 | 点击“+ 发帖”，输入文字，上传图片/视频，选择标签（可选），标注位置，设置可视权限（公开/仅自己可见），点击“发布” | 页面顶部返回和发布按钮；文本框提示“分享你的遛狗日常...”；图片上传支持 3:4 尺寸；标签可多选；位置显示选择控件；可视权限单选控件 |
| 帖子浏览 | 双列瀑布流展示帖子 | 用户上下滑动浏览帖子 | 每个帖子卡片显示用户头像+昵称、标签（非必选）、文字摘要、3:4 图片缩略图、位置（可选）、点赞/评论/收藏数量和按钮 |
| 帖子详情 | 查看帖子的完整内容 | 点击帖子卡片 | 顶部返回按钮，用户头像+昵称+标签，完整文字内容，3:4 图片或视频内容，位置显示（仅公开帖可见），点赞/评论/收藏按钮，评论列表可点赞，评论输入框 |
| 收藏管理 | 用户可管理自己收藏的帖子 | 我的页面 > 我的收藏，切换标签：我收藏的地点 / 我收藏的帖子 | 双列瀑布流展示收藏内容；点击进入详情页；标签切换明显，易用 |
| 可视权限 | 用户可设置帖子公开或私密 | 发帖页选择公开 / 仅自己可见 | 控件明确易用，默认公开；私密帖子不显示在社区广场，仅在个人页面可见 |
| 位置标注 | 用户可添加帖子位置 | 发帖页选择当前或自定义位置 | 位置选择控件，支持自动获取当前地点或手动搜索标注；帖子卡片和详情页显示位置（仅公开帖可见） |

---

## 3. 用户流程

1. **发帖流程**：

   * 进入社区广场 → 点击“+ 发帖” → 输入文字 → 上传图片/视频 → 选择标签（可选） → 标注位置 → 选择可视权限 → 点击“发布” → 成功显示在社区广场或“我的帖子”中（根据权限）

2. **浏览帖子**：

   * 打开社区广场 → 上下滑动双列瀑布流 → 点击卡片查看详情 → 点赞/评论/收藏

3. **管理收藏**：

   * 我的页面 → 我的收藏 → 标签切换（地点/帖子） → 查看详情

---

## 4. UI/UX 设计原则

* **布局**：

  * 社区广场：双列瀑布流，图片3:4，文字摘要长度统一
  * 发帖页：顶部返回+发布按钮，控件分组清晰
  * 收藏页：统一双列展示，标签切换明显

* **视觉**：

  * 简洁、清爽，突出内容
  * 图片/视频缩略图保持 3:4 比例
  * 标签使用浅色背景圆角样式

* **交互**：

  * 点赞、评论、收藏按钮可点击，状态高亮显示
  * 位置显示仅在公开帖可见
  * 可视权限和位置选择控件直观、易操作

* **用户反馈**：

  * 发布成功提示
  * 收藏、点赞、评论状态即时更新
